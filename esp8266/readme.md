# Api

## Publish

### /state

#### On

Switches the lamp state to 'ON'

#### OFF

Switches the lamp state to 'OFF'

### /brightness

#### 1 ... 100

Sets the brightness of the lamp

#### OFF

Switches the lamp state to 'OFF'

### /fade

#### On

Enables fading

#### OFF

Disables fading

### /preset

#### 1 ... 5

Loads given preset number

## Subscribe

You can subscribe to the topics mentioned in 'Publish' section.
